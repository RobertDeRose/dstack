# Implemented features

This index tracks features during close-out and after delivery. Each record states its current reconciliation,
validation, and delivery status.

Each feature record explains:

- delivered user-visible behavior;
- architecture integration;
- operational impact;
- validation evidence;
- intentional changes from the original design;
- related documentation, Beads root, commits, and pull requests.

Planned or partially implemented features remain in [Planned Features](../planned-features.md) and Beads. They are added
here only during close-out.
